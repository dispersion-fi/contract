    VotingController.vy #

    vyper: 0.2.11 author: Dispersion Finance Team license: MIT

    Voting Controller

    For voting, Reaper's from Controller are used.

    Сontrols voting for the distribution of farm token emission by Reaper's.

    Events #

    Vote

    • reaper : address, notIndexed
    • coin : address, notIndexed
    • account : address, notIndexed
    • amount : uint256, notIndexed

    Unvote

    • reaper : address, notIndexed
    • coin : address, notIndexed
    • account : address, notIndexed
    • amount : uint256, notIndexed

    StartVoting

    • delay : uint256, notIndexed

    CommitOwnership

    • admin : address, notIndexed

    ApplyOwnership

    • admin : address, notIndexed

    Methods #

    init #

    type: nonpayable constructor

    Contract constructor

    _controller, _gasTokenCheckList, _farmToken can't be equal ZERO_ADDRESS. owner = msg.sender

    Arguments:

    • _controller: - Address of Controller contract

    • _gasTokenCheckList: - Address of AddressesCheckList contract with gas tokens allowed

    • _farmToken: - Address of FarmToken contract (DSP)

    snapshot #

    type: nonpayable function

    Makes a snapshot and fixes voting result per voting period, also updates historical reaper vote integrals

    Only possible to call it once per voting period

    Arguments:

    • _gasToken: - Gas token address (optional)

    snapshot #

    type: nonpayable function

    Only possible to call it once per voting period

    Arguments:

    • _gasToken: - Gas token address (optional)

    vote #

    type: nonpayable function

    Vote for reaper _reaper using tokens _coin with amount _amount for user msg.sender

    Arguments:

    • _reaper: - Reaper to vote for

    • _coin: - Coin which is used to vote

    • _amount: - Amount which is used to vote

    • _gasToken: - Gas token address (optional)

    vote #

    type: nonpayable function

    Arguments:

    • _reaper: - Reaper to vote for

    • _coin: - Coin which is used to vote

    • _amount: - Amount which is used to vote

    • _gasToken: - Gas token address (optional)

    unvote #

    type: nonpayable function

    Unvote for reaper _reaper and withdraw tokens _coin with amount _amount for _account

    Only possible with unlocked amount

    Arguments:

    • _reaper: - Reaper to unvote for

    • _coin: - Coin which is used to unvote

    • _amount: - Amount which is used to unvote

    • _gasToken: - Gas token address (optional)

    Check for available withdrawal amount from reaper _reaper for tokens _coin for account _account

    Arguments:

    • _coin: - Coin which is used to unvote

    • _account: - Account who is unvoting

    Returns:

    • _0 - Unlocked amount to withdraw

    startVoting #

    type: nonpayable function

    Unlock and start voting mechanism.

    Callable once by controller only. Emit StartVoting(_votingDelay) event. Init equal distribution of votes (1/REAPERS_COUNT).

    Arguments:

    • _votingDelay: - Initial delay defore next voting

    startVoting #

    type: nonpayable function

    Callable once by controller only. Emit StartVoting(_votingDelay) event. Init equal distribution of votes (1/REAPERS_COUNT).

    Arguments:

    • _votingDelay: - Initial delay defore next voting

    transferOwnership #

    type: nonpayable function gas: 37891

    Transfers ownership by setting new owner _futureOwner candidate

    Callable by owner only. Emit CommitOwnership event with _futureOwner

    Arguments:

    • _futureOwner: - Future owner address

    applyOwnership #

    type: nonpayable function gas: 38747

    Applies transfer ownership

    Callable by owner only. Function call actually changes owner. Emits ApplyOwnership event with _owner

    setVotingToken #

    type: nonpayable function gas: 37623

    Set voting token address.

    Callable once by owner only. _votingToken can't be equal ZERO_ADDRESS.

    Arguments:

    • _votingToken: - New address of StrictTransferableToken(ERC20 for voting) contract

    setVotingTokenRate #

    type: nonpayable function gas: 71671

    Set voting token rate and amplifier.

    Callable by owner only. _rate and _rate should be greater 0.
