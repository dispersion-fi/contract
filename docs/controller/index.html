    Controller.vy #

    vyper: 0.2.11 author: Dispersion Finance Team license: MIT

    Controller

    Controls liquidity Reaper's and the emission of coins through the Reapers. Add/remove Reaper from voting/emission.

    Methods #

    init #

    type: nonpayable constructor

    Contract constructor

    _gasTokenCheckList, _farmToken can't be equal ZERO_ADDRESS. owner = admin = msg.sender

    Arguments:

    • _farmToken: - Address of FarmToken contract (DSP)

    • _gasTokenCheckList: - Address of AddressesCheckList contract with gas tokens allowed

    mintFor #

    type: nonpayable function

    Mint available tokens from _reaper for _account.

    _reaper should be added before. startMintFor should be equal True

    Arguments:

    • _reaper: - Address of Reaper contract for minting(claiming)

    • _account: - Address of account for balance query (msg.sender by default)

    • _gasToken: - Gas token address (optional)

    type: nonpayable function gas: 108720

    Add Reaper from Controller.

    Callable by owner only. _reaper can't be equal ZERO_ADDRESS and duplicated.

    Arguments:

    • _reaper: - Address of Reaper contract for adding

    setAdmin #

    type: nonpayable function gas: 36568

    Set address of admin.

    Callable by owner only. _admin can't be equal ZERO_ADDRESS.

    Arguments:

    • _admin: - Address of new admin

    startMinting #

    type: nonpayable function gas: 38629

    Allow to claim tokens rewards.

    Callable by owner only. Emits StartMinting event.

    startEmission #

    type: nonpayable function gas: 6387

    Start emission of FarmToken.

    Callable by owner only. Emits YearEmissionUpdate(INITIAL_YEAR_EMISSION) and StartVoting(_votingDelay) events

    Arguments:

    • _votingController: - Address of VotingController contract

    • _votingDelay: - Initial delay defore next voting

    transferOwnership #

    type: nonpayable function gas: 37891

    Transfers ownership by setting new owner _futureOwner candidate

    Callable by owner only. log YearEmissionUpdate(INITIAL_YEAR_EMISSION)

    Arguments:

    • _futureOwner: - Future owner address

    applyOwnership #

    type: nonpayable function gas: 38747

    Applies transfer ownership

    Callable by owner only. Function call actually changes owner. Emits ApplyOwnership event with _owner
