    init #

    type: nonpayable constructor

    Contract constructor

    owner = msg.sender.

    addAddress #

    type: nonpayable function gas: 39147

    Adds address to list for later verification(check).

    Callable by owner only. _address can't be equal ZERO_ADDRESS.

    Arguments:

    • _address: - Address to add to list

    removeAddress #

    type: nonpayable function gas: 24177

    Remove address from list.

    Callable by owner only. _address can't be equal ZERO_ADDRESS.

    Arguments:

    • _address: - Address to remove from list

    transferOwnership #

    type: nonpayable function gas: 38981

    Transfers ownership by setting new owner _futureOwner candidate

    Callable by owner only. Emit CommitOwnership event with _futureOwner

    Arguments:

    • _futureOwner: - Future owner address

    applyOwnership #

    type: nonpayable function gas: 41137

    Applies transfer ownership

    Callable by owner only. Function call actually changes owner. Emits ApplyOwnership event with _owner
