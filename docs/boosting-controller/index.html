    Methods #

    boost #

    type: nonpayable function

    Boost reward using boost tokens with amount _amount for user msg.sender

    Arguments:

    • _amount: - Amount which is used to boost

    • _lockTime: - Time for locking boost tokens

    • _gasToken: - Gas token for tx cost reduction

    boost #

    type: nonpayable function

    Arguments:

    • _amount: - Amount which is used to boost

    • _lockTime: - Time for locking boost tokens

    • _gasToken: - Gas token for tx cost reduction

    unboost #

    type: nonpayable function

    Unboost for account msg.sender and withdraw all available boost tokens

    Only possible with unlocked amount, withdraw all amount

    Arguments:

    • _gasToken: - Gas token for tx cost reduction

    unboost #

    type: nonpayable function

    Only possible with unlocked amount, withdraw all amount

    Arguments:

    • _gasToken: - Gas token for tx cost reduction

    availableToUnboost #

    type: view function gas: 5065

    Check for available withdrawal amount for account _account

    Arguments:

    • _account: - Account who is unboosting

    Returns:

    • _0 - Unlocked amount to withdraw

    transferOwnership #

    type: nonpayable function gas: 39071

    Transfers ownership by setting new owner _futureOwner candidate

    Callable by owner only. Emit CommitOwnership event with _futureOwner

    Arguments:

    • _futureOwner: - Future owner address

    applyOwnership #

    type: nonpayable function gas: 41227

    Applies transfer ownership

    Callable by owner only. Function call actually changes owner. Emits ApplyOwnership event with _owner
