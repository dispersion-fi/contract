    tokens/StakableERC20.vy #

    vyper: 0.2.11 author: Dispersion Finance Team license: MIT

    Stakable ERC20 Token

    Based on the ERC-20 token standard. USDN for example.

    Mintable and Stakable ERC20 token. balanceOf(account) = balances[account] * p(n) / p(i), p(n) = p(n-1) * (1 + reward / totalSupply). Reward is not stake for funds deposited recently.

    Events #

    Transfer

    • sender : address, indexed
    • receiver : address, indexed
    • value : uint256, notIndexed

    Approval

    • owner : address, indexed
    • spender : address, indexed
    • value : uint256, notIndexed

    Deprecate

    • account : address, indexed

    Reward

    • id : uint256, notIndexed
    • amount : uint256, notIndexed

    deposit #

    type: nonpayable function gas: 159480

    Deposit tokens from other stakable network. Mint _amount tokens for _account.

    Emits a Transfer event from ZERO_ADDRESS to _account with _amount. _account can't be equal ZERO_ADDRESS and _amount must be greater 0. Deposit is forbid if the contract is deprecated. Callable by owner only. Reward is not stake for funds deposited recently (after last stake call).

    Arguments:

    • _account: - Account to mint tokens for

    • _amount: - Amount to mint

    Returns:

    • _0 - Boolean success value

    stake #

    type: nonpayable function gas: 135270

    Stake reward. Mint _reward tokens for all holders immediently (low cost).

    Emits a Reward event. _reward must be greater 0. Deposit is forbid if the contract is deprecated. Callable by owner only. Reward is not stake for funds deposited recently (after last stake call) and first call.

    Arguments:

    • _reward: - Amount to mint reward

    Returns:

    • _0 - Boolean success value

    withdraw #

    type: nonpayable function gas: 121450

    Withdraw tokens back to other stakable network. Burn all token balance for _account.

    Emits a Transfer event from _account to ZERO_ADDRESS for all token balance. Deposit is forbid if the contract is deprecated. Callable by owner only.

    Arguments:

    • _account: - Account for burn all tokens

    Returns:

    • _0 - Boolean success value

    approve #

    type: nonpayable function gas: 39711

    Approves allowance from msg.sender to _spender address for _amount of tokens

    ERC20 function. Emits a Approval event with msg.sender, _spender, _amount.

    Arguments:

    • _spender: - Allowed account to send tokens from msg.sender

    • _amount: - Allowed amount to send tokens from msg.sender

    Returns:

    • _0 - Boolean success value

    Arguments:

    • _recipient: - Account to send tokens to

    • _amount: - Amount to send

    Returns:

    • _0 - Boolean success value

    transferFrom #

    type: nonpayable function gas: 281985

    Transfers _amount of tokens from _sender to _recipient address

    ERC20 function. Allowance from _sender to msg.sender is needed. Emits a Transfer event with _sender, _recipient, _amount. _sender and _recipient can't be equal ZERO_ADDRESS

    Arguments:

    • _sender: - Account to send tokens from

    • _recipient: - Account to send tokens to

    • _amount: - Amount to send

    Returns:

    • _0 - Boolean success value
